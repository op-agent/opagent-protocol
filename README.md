# OpAgent Protocol

Operation Agent Protocol (OpAgent Protocol) is a framework for building multi-agent systems. It defines the core protocol and SDKs for composing agents, skills, tools, and subagents into interoperable agent runtimes.

OpAgent Protocol is adapted from and builds on the excellent [modelcontextprotocol/go-sdk](https://github.com/modelcontextprotocol/go-sdk). We are grateful to the Model Context Protocol project and its contributors for the foundational Go SDK design and implementation that made this work possible.

## Go SDK

The Go SDK lives in [`go-sdk`](./go-sdk).

```bash
cd go-sdk
go test ./...
```

Module path:

```text
github.com/op-agent/opagent-protocol/go-sdk
```
